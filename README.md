# UK Crime & Socioeconomic Analysis

An analysis of crime trends across England using open data from police.uk 
and the ONS, exploring how crime patterns relate to deprivation, population 
density, and geography between 2018 and 2023.

---

## What this project looks at

- How crime rates and types have shifted post-pandemic
- Whether there's a measurable relationship between deprivation indices and crime rates
- Differences in crime patterns between urban and rural areas

---

## Data sources

- [police.uk Open Data](https://data.police.uk/)
- [ONS Deprivation Statistics](https://www.ons.gov.uk/)
- [ONS Population Estimates](https://www.ons.gov.uk/)

---

## Tools used

Python (pandas, numpy, seaborn, plotly), Jupyter Notebooks, SQL

---

## Running the project
```bash
git clone https://github.com/yourusername/uk-crime-analysis.git
cd uk-crime-analysis
pip install -r requirements.txt
```

Notebooks are numbered and intended to be run in order.

---

## Repo structure

- `data/` — raw and processed datasets (raw files are gitignored)
- `notebooks/` — analysis notebooks
- `src/` — helper functions and data loading scripts
- `outputs/figures/` — saved visualisations
